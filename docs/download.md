## Installation

### macOS

1. Telechargez le fichier `.dmg`
2. Ouvrez le fichier telecharge
3. Glissez Comptine dans le dossier Applications
4. Au premier lancement, faites clic droit > Ouvrir pour autoriser l'application

### Windows

1. Telechargez le fichier `.exe`
2. Executez l'installateur
3. Suivez les instructions d'installation
4. Lancez Comptine depuis le menu Demarrer

## Code source

Comptine est un logiciel libre. Le code source est disponible sur [GitHub]({{ site.github_repo }}).

Vous pouvez compiler l'application vous-meme en suivant les instructions du fichier README.
